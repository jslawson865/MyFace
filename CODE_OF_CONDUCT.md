# Code of Conduct

MyFace is designed around small communities, intentional invitations, and accountable participation.

Expected behavior:

- build useful things,
- respect local node rules,
- do not harass, spam, scam, or brigade,
- do not weaponize reports or downvotes,
- use moderation tools transparently and proportionally.

The project distinguishes between popularity, trust, risk, and renown. High engagement does not excuse abusive conduct.
