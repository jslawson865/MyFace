- backend-enforced invites,
- User 0 default friend,
- profiles,
- friends,
- bulletins,

- node settings,

- signed public profile metadata,
- public group/project metadata,
- public bulletin metadata,
- cross-node reputation summaries,
- block/defederation notices.
